AirAware is a Flask-based web application focused on environmental sustainability.
It helps users understand Air Quality Index (AQI) levels and pollution sources
in Indian cities.

## Problem Statement
Air pollution is a major issue in India caused by industrial emissions,
vehicle pollution, construction dust, and deforestation.
High AQI levels affect human health and environment.

## Solution
This project allows users to:
- Enter city name
- Enter AQI value
- Select pollution source

Based on the AQI value, the system classifies air quality
as Good, Moderate, Poor, or Severe.

## Features
- Simple user interface
- AQI classification logic
- Flask backend
- Educational purpose

## Technologies Used
- Python
- Flask
- HTML
- CSS
- GitHub

## How to Run the Project
1. Install Python
2. Install Flask using: pip install flask
3. Run the app using: python app.py
4. Open browser and go to: http://127.0.0.1:5000/
